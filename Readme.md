# states

  An object containing states.

## Installation

    $ component install component/states

## API

```js
module.exports = {
  AL: 'Alabama',
  AK: 'Alaska',
  AZ: 'Arizona',
  AR: 'Arkansas',
  CA: 'California',
...
```
